하이퍼링크

하이퍼링크 사용하기

a 태그

a태그는 링크를 만드는 태그이며 많이 사용되는 태그중 하나이다
a는 anchor(닻)의 약자로서 기본문법은 아래와 같다
<a href="address">Label</a>
hre속성을 사용하여 연결할 주소를 지정할 수 있다(hypertext reference)
href 속성 값에 웹페이지, 이미지, 동영상, 파일 등의 주소를 넣을 수 있다.( 웹브라우저의 설정에 따라 다르다 ) 따옴표를 반드시 사용한다
Label은 표시할 텍스트를 의미한다.



target 속성

target속성은 링크를 어디에서 열지 결정한다. traget 속성의 값으로 넣을 수 있는 값은 다음과 같다

  1. _self: 기본값으로 현재 페이지에서 링크 처리
  2. _blank: 새창 또는 새탭에서 링크 처리
  3. _parent, _top: 웹페이지를 frameset으로 구성했을때 사용

_self속성 적용(현재 페이지에서 열기) hoho의 블로그
_blank속성 적용(새로운 탭에서 열기) hoho의 블로그



download 속성

download를 추가하면 웹브라우저의 속성에 상관없이 링크대상을 다운로드 한다
<a href="address" download >Label</a>
위에처럼 하면 링크를 눌렀을때 다운로드가 된다.



문서 내 특정 위치로의 링크

href 속성의 값 앞에 #을 붙이면 해당 id를 가진 요소로 이동할 수 있다.
<a href="#tag1">Label</a>
위 처럼 작성하게 되면 id의 속성값으로 tag1을 갖는 요소로 이동하게 된다.
아래는 id를 이용하여 맨 위로 이동할 수 있게 링크를 작성하였다 클릭해보자
맨 위로 이동



title 속성

 

title속성은 해당 링크에 마우스를 가져다 대면 링크에 대한 설명을 표시해준다.

사용법은 아래와 같다

<a href="address" title=" this is my blog">Label</a>
