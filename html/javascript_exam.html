  • html의 구조 만들기
  • 자바스크립트 따라하기

  1. getElementById() 메소드

    1. 내용 바꾸기( innerHTML )

      2. input 태그를 이용하여 버튼을 만들고 태그안에 onclick 속성을 줌으로써 동적으로 동작할 수 있다.

      Example

      onclick='document. getElementById ("change_text"). innerHTML = "Change Text!!"';

      before changing text

    2. 이미지 바꾸기 ( src )

      3. 방법은 1번과 비슷하다. 하지만 이번에는 innerHTML 속성이 아닌 src속성을 이용한다
      src의 속성으로 이미지의 '경로/이미지이름'을 넣어주면 이미지를 바꿀수 있다.

      Example

      onclick='document. getElementById ("change_text"). src = "pic_bulbon.gif"';

    3. CSS 바꾸기 ( style )

      4. 이번에는 style 속성(attribute)을 바꿔보려고 한다. style속성 중에서도 글씨크기를 바꾸려한다
      글씨크기를 바꾸는 속성은 fontSize이다

      Example

      onclick='document. getElementById ("change_Text2"). style.fontSize = "30px"';

      before changing text

    4. 내용 숨기고 표시하기

      5. 이번에도 style속성을 이용하려한다. style의 속성중에 하나인 display 속성을 이용하여 글씨를 나타나거나 사라지게 할 수 있다.

      Example

      onclick='document. getElementById ("change_text"). src = "pic_bulbon.gif"';

      Visible text!!!
