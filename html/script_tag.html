  1. Script tag 사용하기

    1. script tag

      2. script tag를 이용하 앞에서 배운 속성을 사용할 수 있다. 따로 script태그를 두어서 구별하기 편하다는 장점이 있다.
      script tag는 body와 head 부분에 위치하며 두군데 모두 위치할 수 있다.
      하지만 tag를 사용할때 지정할 id보다 태그가 밑에 있어야 하므로 태그를 제일 하단에 두는것이 좋다

      Example

      <script>
      onclick='document. getElementById ("change_text"). innerHTML = "Change Text!!"';
      </script>

      before changing text

    2. function 이용하기

      3. 똑같이 style태그를 이용하려고 한다. 하지만 이번에는 script 태그를 head에 위치시켜 function을 이용해서 함수를 불러다 쓰려고한다.
      함수를 쓰지 않으면 사용하려는 태그의 밑에 script 태그가 위치해야만 동작하지만 이렇게 하면 어디에 위치하건 사용가능하다
      예시는 head에서 사용하는걸 했지만 body에 script 태그를 두고 function을 만들어서 사용할 수 있다.

      Example

      < head >
      <script>
      function myFunctoin() {
        document. getElementById ("change_text"). innerHTML = "Change Text!!";
      </script>
      </ head >

      <body>
      <p id="demo">A Paragraph</p>
      <input type="button" onclick="myFunction()">Try it</input>
      <body>
